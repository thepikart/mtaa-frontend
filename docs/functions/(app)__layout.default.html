    • default(): Element

      Root layout component that defines the navigation Stack for the application.

      Responsibilities

      1. Authentication gate – if the user is not logged in, immediately redirects to the /login screen.
      2. Push notifications – once the user is present, requests a push‑token and registers it with the backend via useUserStore.
      3. Offline queue sync – after mount, loads any locally stored requests and attempts to synchronise them with the server.
      4. In‑app message handling – listens for foreground Firebase Cloud Messaging (FCM) notifications and shows them in a toast; when the toast is tapped it deep‑links to the related event.

      Returns Element

      A React element configuring the navigation stack for the Expo Router.

      Remarks

      This component is meant to be mounted at the very top of the tree (similar to an App component created by expo prebuild). Because it mounts only once, it is safe to place long‑lived listeners (e.g., FCM subscription) here.
